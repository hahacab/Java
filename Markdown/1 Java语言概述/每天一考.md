## 1.JDK,JRE,JVM三者之间的关系，以及JDK、JRE包含的主要结构有哪些

JDK = JRE + Java的开发工具（javac.exe,java.exe,javadoc.exe等）
JRE = JVM + Java核心类库

## 2.为什么要配置path环境变量？如何配置？

为了在任何的文件路径下运行javac,java等开发工具。

JAVA_HOME = bin的上一层目录
path = %JAVA_HOME%\bin
